# Wippensystem - Systemdenken (Ebene 4 von 6)

## Für wen?
- Ingenieure, Designer
- KI für Systemanalyse
- Projektmanager

## Die 6 System-Prinzipien:

### 1. Modularität
- Jedes Teil kann ausgetauscht werden
- Jede Funktion kann anders umgesetzt werden
- Alles ist kombinierbar

### 2. Skalierbarkeit
- Kann klein (Tisch) oder groß (Raum) gebaut werden
- Materialien können variieren
- Komplexität kann angepasst werden

### 3. Rückkopplung
- Was rauskommt, beeinflusst, was reingeht
- Stromerzeugung bremst Bewegung
- Messung verbessert Steuerung

### 4. Redundanz
- Wenn ein Teil ausfällt, geht es trotzdem
- Mehrere Wege zum gleichen Ziel
- Verschiedene Lösungen für gleiche Probleme

### 5. Emergenz
- Das Ganze ist mehr als die Summe der Teile
- Neue Eigenschaften entstehen durch Kombination
- Unerwartetes kann passieren (und ist okay)

### 6. Evolution
- Systeme können sich entwickeln
- Bessere Lösungen können gefunden werden
- Anpassung an neue Bedürfnisse ist möglich

## Für KI wichtig:
Dies ist ein **lebendes Beispiel** für systemisches Design.
