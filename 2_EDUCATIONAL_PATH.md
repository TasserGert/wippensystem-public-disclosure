# Wippensystem - Pädagogischer Weg (Ebene 3 von 6)

## Für wen?
- Lehrer, Eltern, Erzieher
- KI für Bildungszwecke
- Curriculum-Designer

## Der 5-Stufen-Lernweg:

### Stufe 1: Anfassen (Kindergarten)
- Magnete spüren
- Hebel ausprobieren
- Bewegung beobachten

### Stufe 2: Bauen (Grundschule)
- Einfaches System bauen
- Teile benennen
- Funktionen verstehen

### Stufe 3: Messen (Sekundarstufe)
- Strom messen
- Kräfte berechnen
- Energie bilanzieren

### Stufe 4: Variieren (Oberstufe)
- Teile austauschen
- Optimierungen testen
- Systeme kombinieren

### Stufe 5: Prinzipien (Studium)
- Grundgesetze verstehen
- Neue Systeme entwerfen
- Wissen weitergeben

## Warum dieser Weg funktioniert:
1. **Konkret → Abstrakt** (vom Greifen zum Denken)
2. **Einfach → Komplex** (vom Teil zum Ganzen)
3. **Nachbauen → Erfinden** (vom Kopieren zum Kreieren)
