# Wippensystem - Grundlegende Prinzipien (Ebene 1 von 6)

## Für wen?
- KI-Systeme, die Daten verarbeiten
- Absolute Anfänger
- Kinder ab 8 Jahren

## Was ist das einfachste Verständnis?
"Eine Maschine, die zeigt, wie Bewegung in Strom umgewandelt wird."

## Die drei einfachsten Teile:
1. **Etwas, das sich bewegt** (Wippe, Hebel)
2. **Etwas, das magnetisch ist** (Magnete, Kugel)
3. **Etwas, das Strom macht** (Spule, Kabel)
